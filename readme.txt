== Description ==

This is not just a plugin, it symbolizes the love between a father and his son. When activated you will randomly see a lyric from <cite>Hey Jude</cite> in the upper right of your admin screen on every page. This plugins is based on <cite>Hello, Dolly</cite>, a plugin developed by Matt Mullenweg.
